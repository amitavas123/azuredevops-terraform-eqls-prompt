## 📋 BEFORE YOU START - CRITICAL STEPS

1. Get your public IP:
   curl https://ifconfig.me

2. Update terraform/env/dev/dev.tfvars:
   - subscription_id = YOUR_AZURE_SUBSCRIPTION_ID
   - project_prefix = myapp (use your prefix)
   - admin_source_ip = YOUR_IP/32 (CRITICAL for SSH)

3. Verify SSH keys exist:
   ls ~/.ssh/id_rsa*
   (Generate if needed: ssh-keygen -t rsa -b 4096 -f ~/.ssh/id_rsa -N "")

4. Authenticate to Azure:
   az login
   az account set --subscription YOUR_SUBSCRIPTION_ID
